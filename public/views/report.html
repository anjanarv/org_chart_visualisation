Org Hierarchy is a web app that gives a visual representation of a team structure in an organization.
Test data used here to display the hierarchies are stored in JSON formats.

Below is the tech stack used to build this app:

Web app framework used to build Org Hierarchy

Express.js

Library used to perform Graph Analysis

Cytoscape.js

Library used to visualise the analysed graph data in a hierarchical representation

VisJS
